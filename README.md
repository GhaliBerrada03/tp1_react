# 🚀 Projet React - TP

## 📌 Description

Ce projet est une application React simple qui utilise plusieurs composants :

* HelloWorld
* Greeting
* Profil
* Voiture
* ListeCourses

Il permet de pratiquer les bases de React :

* composants
* props
* affichage dynamique

---

## 🛠️ Installation

1. Installer les dépendances :

```bash
npm install
```

2. Lancer le projet :

```bash
npm start
```

3. Ouvrir dans le navigateur :

```
http://localhost:3000
```
